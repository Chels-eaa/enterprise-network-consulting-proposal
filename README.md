# Enterprise Network Consulting Proposal

## Overview

This project involved designing a secure and scalable enterprise network solution for a fictional innovation hub. The objective was to create a reliable network infrastructure that supports multiple departments, secure communication, wireless connectivity, and future growth while following industry best practices.

---

## Project Objectives

- Design a secure enterprise network
- Implement VLAN segmentation for different departments
- Plan network redundancy and high availability
- Improve network security
- Support wired, wireless, and IoT devices
- Create a scalable infrastructure for future expansion
